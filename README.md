# 📊 Netflix User Behavior & Churn Analysis

> Analyzing user behavior and churn patterns in a Netflix-like subscription dataset to provide actionable insights for retention strategies.

---

## 🔍 Project Overview
This project examines factors influencing user churn, using engagement metrics, subscription types, and activity patterns to identify high-risk users and improve retention.

---

## 📁 Dataset
- **Source:** Kaggle  
- **Name:** Netflix User Watching Behavior Dataset  
- **Records:** 50,000 users  
- **Features:** Demographics, subscription details, engagement metrics, churn indicators

---

## 🎯 Objectives
- Analyze churn patterns  
- Identify high-risk users  
- Compare churn across subscription types  
- Understand engagement impact on churn  

---

## 🛠 Tools
- Tableau Public (Visualization)  
- Excel / CSV (Data Source)

---

## 🔑 Key Metrics
- **Churn Rate:** `AVG(Churned Users)` (1 = churned, 0 = active)  
- **Engagement Score:** Weighted metric: Watch Time 40%, Sessions/Week 30%, Completion Rate 30%  
- **Recency Segmentation:** Active ≤7 days, At Risk 8–30 days, Dormant 30+ days

---

## 📊 Dashboard Highlights
**KPIs:** Total Users, Churn Rate, Engagement Score, Retention Rate  
**Visuals:** Churn by Subscription, Churn by Country, Device Usage, Engagement vs Churn, Recency vs Churn  

---

## 🔑 Key Insights
- **Subscription Behavior:** Standard users churn the most in total; Premium users have the highest churn rate  
- **Engagement:** Low engagement → high churn risk  
- **Recency:** Dormant users most likely to churn  
- **Device Usage:** Mobile is the preferred streaming device

---

## 💡 Recommendations
- Retention strategies for Premium users  
- Target low-engagement users with personalized content  
- Re-engage dormant users via notifications/promotions  
- Optimize mobile experience
